## 二、短期目标（1-2 周）

### 2.1 API Mock Server
- **目标：** 让 Console 的所有列表/详情页能真正增删改查
- **方案：** json-server 或 MSW（Mock Service Worker）
- **价值：** 演示效果从"静态展示"升级为"可交互产品"

### 2.2 E2E 测试基础
- **目标：** 覆盖登录 + 核心导航 + 模型 CRUD
- **方案：** Playwright
- **价值：** 代码质量保障，CI/CD 准备

---

## 三、中期目标（1-2 月）

### 3.1 国际化框架
- **目标：** 中英双语切换
- **方案：** vue-i18n
- **价值：** 出海 + 国际开发者

### 3.2 组件展示站点
- **目标：** 类似 shadcn-vue 官网的组件文档站
- **方案：** VitePress / Storybook
- **价值：** 开源推广 + 降低使用门槛

### 3.3 CLI 脚手架
- **目标：** `npx create-shadcn-vue-ui` 一键初始化
- **方案：** Node.js CLI + prompts
- **价值：** 30 秒上手体验

---

## 四、长期愿景（3-6 月）

### 4.1 行业模板扩展
- 医疗后台模板

### 4.2 插件生态
- 真正可安装的 Prompt / MCP / Skill 插件

### 4.3 Pro 版本
- 高级组件（数据表格 / 拖拽看板 / 工作流编辑器）
