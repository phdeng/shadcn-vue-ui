## 执行序列

### 1. 从项目类型 CSV 加载指导

基于步骤 2 识别的项目类型（web_app、mobile_app、api_backend 等）：

加载相应的需求和问题指南。

### 2. 进行项目类型特定发现

根据项目类型进行对话：

**如果是 Web 应用：**
- 支持的浏览器/设备
- 性能需求
- 离线功能
- 存储需求

**如果是移动应用：**
- 支持的平台（iOS、Android）
- 离线功能
- 本地权限
- 后台处理

**如果是 API 后端：**
- API 样式（REST、GraphQL）
- 认证/授权
- 速率限制
- 版本管理
